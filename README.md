# COACHTECH Pro 開発パーツプロジェクト

## 必要なインストール
docker desktop 及び nodeをインストール

## セットアップ
```bash
git clone git@github.com:kaishu16/docker-laravel-parts.git
cd docker-laravel-parts
make install
```

## 機能パーツブランチ作成方法
mainから開発ブランチを作成
その際機能を示すブランチ名とする。

例) 認証機能パーツのブランチを作成
git checkout -b auth

## 機能パーツ作成方法
- 記述は全てsrc/package/src/ブランチ名　ディレクトリの配下に作成してください。
